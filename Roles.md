1- Tenats manager

* managing  all tenats , creating new tenat , configure its theme.

2- Tenat Admin

* Manage one tenat , the ability to reach its statistics and tenats info like students, enrollements, teachers, subjects.
3- Tenat Support

4- Tenat Course Manager

* Approve the course content , review its videos and exams and contact the teacher for refinements and tech assistant.

5- Course Worker

* can submit video courses (should be reviewed by tenat course manager), answer questions in the coures community and video comments, make exams to be approved by the teacher and then by the Course Manager, deactivate enrollements.

6- Teacher

* approve exams from worker, write exams and can access info about the course, enrollements

* correct exams from students if it is handwritten
7- Library Manager

* the ability to make new codes for courses , retreive code for a course , deactivate enrollements and see the statistics about their library.

8- Library Worker

* only retreiving codes and generate new ones

9- Library Accountant

* Audit Library Changes,

* Attach, Deattach Courses For the library to have the ability to make codes for theme

9- Tenat Support Employee

* the ability to remove comments , answer issues , for a community and tenat
* Deals with Tenat Admin to manage Teacher Accounts, Subjects, Grades, and other tenat related issues.
