## Student
- As a student, I can register and log in using my phone number or email.
- As a student, I can view all available courses categorized by subject and teacher.
- As a student, I can redeem a course access code purchased from a library.
- As a student, I can buy quizzes or courses using my earned or purchased tokens.
- As a student, I can watch video lessons with a reliable streaming player.
- As a student, I can download course materials if allowed.
- As a student, I can take quizzes of all question types (MCQ, text, math, etc.).
- As a student, I can see my results and progress analytics.
- As a student, I can ask questions in the community area for each course.
- As a student, I can answer questions and earn badges for participation.
- As a student, I can rate courses and teachers.
- As a student, I can send feedback or report problems to the support team.

## Teacher
- As a teacher, I can create, edit, delete courses and upload videos, materials, and quizzes.
- As a teacher, I can assign co-workers with limited permissions to assist in uploading or managing videos.
- As a teacher, I can track student progress and quiz results
- As a teacher, I can see analytics for course views, quiz success rates, and engagement
- As a teacher, I can communicate with students via the course community or private messages
- As a teacher, I can withdraw earnings or track payouts from course sales
- As a teacher, I can respond to student questions in the course community
- As a teacher, I can be a student in another course if enrolled

## Teacher Co-worker
- As a co-worker, I can upload and delete videos/materials for the courses I’m assigned to.
- As a co-worker, I can answer questions sent to the teacher and send him notifications to answer if something is not relatable
- As a co-worker, I can help moderate the teacher’s community discussions.
- As a co-worker, I cannot change pricing or payout settings for the teacher.
- As a co-worker, I cannot delete entire courses or students.

## Admin
- As an admin, I can view all users, roles, and activities.
- As an admin, I can assign or revoke roles (teacher, moderator, library worker, etc.).
- As an admin, I can see all changes in the library system for fraud prevention.
- As an admin, I can approve or reject new courses before publishing.
- As an admin, I can view platform statistics (course sales, active users, performance).
- As an admin, I can monitor community discussions for rule violations.
- As an admin, I can suspend users or flag suspicious behavior.
- As an admin, I can export reports of financials, logs, and user activity.
- As a library manager, I can view code usage statistics.

## Library Manager
- As a library manager, I can generate unique access codes for each course.
- As a library manager, I can track sold codes.
- As a library manager, I can monitor library workers and their activities.

## Library Worker
- As a library worker, I can sell codes to students offline and mark them as sold.
- As a library worker, I can see sales history and report suspicious requests.
- As a library worker, I can revoke a code if it was sold incorrectly.

## Support Employee
- As a support employee, I can view support tickets from teachers and students.
- As a support employee, I can respond to problems (e.g., video not loading, code not redeeming).
- As a support employee, I can contact teachers or admins for unresolved cases.

## Moderator
- As a moderator, I can review flagged community posts.
- As a moderator, I can delete offensive or spam content.
- As a moderator, I can ban temporarily users from community discussions.
