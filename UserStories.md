🧑‍🎓 Student User Stories
🎯 Onboarding & Access

As a student, I want to register and join my school/university tenant so that I access only my institution’s content.

As a student, I want to log in using a simple, low-bandwidth-friendly flow so that I can access the platform reliably.

As a student, I want to redeem an access code bought from a library so that I can unlock a course without online payment.

As a student, I want to see which courses are locked or unlocked so that I know what I can access.

📚 Learning & Courses

As a student, I want to browse available courses in my tenant so that I can choose what to study.

As a student, I want to watch course videos with adaptive quality so that I can study even on slow internet.

As a student, I want to resume a video from where I stopped so that I don’t lose progress.

As a student, I want to download lesson notes (if allowed) so that I can study offline.

🧮 Quizzes & Exams

As a student, I want to take quizzes with MCQ, True/False, Text, and Math answers so that exams match the national curriculum.

As a student, I want to see my quiz score immediately so that I understand my performance.

As a student, I want to review correct and incorrect answers so that I learn from mistakes.

As a student, I want to retake practice quizzes if allowed by the teacher.

📊 Progress & Analytics

As a student, I want to see my progress per course so that I know how close I am to completion.

As a student, I want to see my ranking on leaderboards so that I feel motivated.

As a student, I want to earn badges and tokens so that learning feels rewarding.

💬 Community & Collaboration

As a student, I want to ask questions under a specific course so that I get help from peers and teachers.

As a student, I want to upvote helpful answers so that good content surfaces.

As a student, I want to report inappropriate content so that the community stays safe.

🧑‍🏫 Teacher User Stories
🏫 Tenant & Profile

As a teacher, I want to belong to a specific school/university tenant so that I teach only my students.

As a teacher, I want to manage my profile and qualifications so that students trust my content.

📚 Course Creation & Management

As a teacher, I want to create courses and assign them to a grade and subject.

As a teacher, I want to upload videos and materials so that students can learn asynchronously.

As a teacher, I want to organize content into lessons and chapters so that learning is structured.

As a teacher, I want to publish or unpublish content so that I control availability.

🧮 Quiz & Assessment

As a teacher, I want to create quizzes with different question types so that exams match real tests.

As a teacher, I want to include math formulas using LaTeX so that scientific subjects are supported.

As a teacher, I want to control quiz timing and attempts so that assessments are fair.

📊 Student Tracking

As a teacher, I want to see student performance analytics so that I identify weak areas.

As a teacher, I want to export grades so that I can use them offline if needed.

As a teacher, I want to see which students are inactive so that I can intervene.

💬 Community Interaction

As a teacher, I want to answer student questions so that I guide discussions.

As a teacher, I want to pin or mark correct answers so that students trust official responses.

🧑‍🏫 Teacher Co-Worker User Stories

As a teacher co-worker, I want to upload content on behalf of a teacher so that work is shared.

As a teacher co-worker, I want to manage quizzes but not publish them so that final approval stays with the teacher.

As a teacher co-worker, I want restricted permissions so that I cannot affect payments or students.

🏫 Tenant Admin (School / University Admin)
🏗️ Tenant Management

As a tenant admin, I want to manage my school’s profile and branding.

As a tenant admin, I want to enable or disable features per tenant so that I control usage.

👥 User & Role Management

As a tenant admin, I want to create and assign roles to users within my tenant.

As a tenant admin, I want to invite teachers and staff so that onboarding is easy.

As a tenant admin, I want to deactivate users if needed for security.

📊 Reports & Oversight

As a tenant admin, I want to see usage and revenue reports so that I track performance.

As a tenant admin, I want to view audit logs for my tenant so that I detect misuse.

🧑‍💼 Super Admin (Platform Owner)

As a super admin, I want to manage all tenants so that the platform scales safely.

As a super admin, I want to define global roles and permissions so that access is consistent.

As a super admin, I want to view system-wide analytics so that I make business decisions.

As a super admin, I want to suspend a tenant if fraud or abuse is detected.

📚 Library Manager User Stories

As a library manager, I want to generate prepaid access codes so that students can buy courses offline.

As a library manager, I want to assign codes to specific courses or bundles.

As a library manager, I want to track sold vs unused codes so that revenue is controlled.

As a library manager, I want to revoke unused codes if needed.

📚 Library Worker User Stories

As a library worker, I want to redeem codes for students so that access is granted instantly.

As a library worker, I want limited permissions so that I cannot generate or modify codes.

As a library worker, I want to verify code validity quickly so that sales are efficient.

🧑‍💻 Support Employee User Stories

As a support employee, I want to view user accounts so that I can resolve issues.

As a support employee, I want to reset access or reassign courses so that problems are solved fast.

As a support employee, I want to see support tickets linked to users and tenants.

🛡️ Moderator User Stories

As a moderator, I want to review reported content so that discussions remain respectful.

As a moderator, I want to edit or remove inappropriate posts.

As a moderator, I want to ban users temporarily from discussions if they violate rules.

🔍 Auditor User Stories

As an auditor, I want read-only access to all logs so that fraud is detectable.

As an auditor, I want to track payment-related actions so that financial integrity is preserved.

As an auditor, I want immutable logs so that records cannot be altered.
